# Git
# Comandos importantes 

```bash
git init - #Inicializa um novo repositório Git em um diretório.

git clone - #Clona um repositório Git existente para o seu sistema local.

git add - #Adiciona arquivos preparando para o commit.

git commit -m - #Cria um novo commit com os arquivos com uma mensagem de commit.

git status - #Exibe o estado atual do seu repositório.

git log - #Mostra o histórico de commits do repositório.

git branch - #Lista todas as ramificações (branches) no repositório e indica qual ramificação está atualmente em uso.

git checkout - #Altera para uma ramificação específica.

git merge - #Une uma ramificação com a ramificação atual.

git pull - #Atualiza o repositório local com as mudanças do repositório remoto.

git push - # Envia os commits locais para o repositório remoto.

git remote - #Exibe informações sobre repositórios remotos configurados.

git fetch - #Obtém informações sobre repositórios remotos.

git reset - #Permite reverter commits e mover o ponteiro HEAD para uma revisão anterior.

git stash - #Salva temporariamente as mudanças não commitadas em um espaço de trabalho limpo.

git diff - #Mostra as diferenças entre o seu diretório de trabalho e o último commit.

git tag - #Gerencia tags (rótulos) para marcar commits específicos como releases ou versões.

git remote add - #Adiciona um novo repositório remoto com um nome específico.
```
